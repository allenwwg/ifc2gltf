# ifc2gltf
convert ifc file to gltf2 file  

## Prerequisites
Git  
CMake  
Windows: Visual studio 2017 with c++ toolset  
2nix: GCC 5.5 or newer  

## Dependencies
Boost  
Open Cascade  
IfcOpenShell

### Compiling on Cent OS
